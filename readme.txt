Для того, чтобы успешно загрузить базу данных с данными из скрипта, нужно:
1. Открыть скрипт(название файла 'ms.sql') в Microsoft SQL Server Managnemt Studio, если есть файл с названием 'my.sql', то его нужно открыть в MySQL Workbench.
2. В первой строке (USE [10180015]) в скобках нужно написать имя своей базы данных, если она сущестувет.
(Создать базу данных: CREATE DATABASE [db*] * - Вместо db нужно вписать своё название)
3. Выполнить скрипт, нажав на кнопку "Выполнить".
